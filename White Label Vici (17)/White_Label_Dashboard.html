VICIdial Architecture

Understanding the architecture is crucial for successful white labeling:

Frontend (PHP + JS)

Agent and admin web interfaces

Location: /srv/www/htdocs/

Backend (Perl)

AGI scripts, cron jobs, processing

Location: /usr/share/astguiclient/

Database (MySQL)

Call logs, campaigns, agents, leads

Database: asterisk

PBX (Asterisk)

Telephony platform, call control

Location: /etc/asterisk/

Critical Files for Branding

File Path Purpose Difficulty
/agc/images/vdc_tab_vicidial.gif Agent interface logo Easy
/vicidial/vicidial_admin_web_logo.gif Admin interface logo Easy
/agc/css/custom.css Custom styles Medium
/agc/options.php Agent configuration Medium
/vicidial/admin_header.php Admin page header Medium
/agc/vicidial.php Core agent interface Hard

Step 1: Logo Replacement

The easiest and most visible change - replace the VICIdial logos with your own.

Requirements

  • Logo dimensions: Exactly 220 x 60 pixels
  • Format: GIF (required for compatibility)
  • File access: SSH or FTP to web server

Commands

# Backup original logos cp /srv/www/htdocs/agc/images/vdc_tab_vicidial.gif \ /srv/www/htdocs/agc/images/vdc_tab_vicidial.gif.backup cp /srv/www/htdocs/vicidial/vicidial_admin_web_logo.gif \ /srv/www/htdocs/vicidial/vicidial_admin_web_logo.gif.backup # Replace with your logos cp your_agent_logo.gif /srv/www/htdocs/agc/images/vdc_tab_vicidial.gif cp your_admin_logo.gif /srv/www/htdocs/vicidial/vicidial_admin_web_logo.gif # Set correct permissions chmod 644 /srv/www/htdocs/agc/images/vdc_tab_vicidial.gif chmod 644 /srv/www/htdocs/vicidial/vicidial_admin_web_logo.gif

Step 2: Custom CSS

Apply your brand colors and styling across the interface.

Create Custom CSS File

/* /srv/www/htdocs/agc/css/custom.css */ :root { --primary-color: #0066CC; /* Your brand blue */ --secondary-color: #FF6600; /* Your brand orange */ --accent-color: #00CC66; /* Your brand green */ } #vicidial_header { background: linear-gradient(135deg, var(--primary-color) 0%, var(--secondary-color) 100%) !important; } .button, input[type="submit"] { background-color: var(--primary-color) !important; border-color: var(--primary-color) !important; } table.vicidial_table th { background-color: var(--primary-color) !important; color: white !important; }
Pro Tip: Use !important to override existing VICIdial styles.

Step 3: Configuration

Customize system settings and behavior.

Database Updates

-- Update system branding UPDATE system_settings SET sounds_web_server = 'https://yourcompany.com', outbound_email_from = 'noreply@yourcompany.com'; -- Update campaign descriptions UPDATE vicidial_campaigns SET campaign_description = CONCAT( 'Your Company - ', campaign_description );
